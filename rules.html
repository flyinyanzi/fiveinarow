技能五子棋｜技能规则说明

综艺模式 手机友好

通用规则

  • 基础为 15×15 五子棋，先连五者胜。
  • 每回合至多使用一次主动技能；若已落子，本回合不可再用技能。
  • 反应技能（擒拿 / 调虎离山 / 东山再起 / 手刀 / 两极反转）仅在窗口内出现，不占用回合“本回合一次技能”的次数。
  • 技能结算期间（准备 / 反应窗口 / 口令窗口），禁止落子与其他操作。
  • 部分技能需对方已有落子或场上有对方棋子，否则不可发动，会提示“对方还没有落子，无计可施哦”。
名词与提示
  • 准备阶段：技能已宣告但可被反制的短时间，例如“梅开二度”后给对手的 3 秒“擒拿”窗口。
  • 反应窗口：仅在特定 3 秒内，反制类按钮会出现在指定玩家的技能区。
  • 浅灰按钮：当前不可用（时机未到/被跳过/已落子/窗口期等）。
  • 深灰 + ✅：一次性技能已使用。

技能列表

飞沙走石

主动 需对手上一步
  • 把对手上一手的棋子移出棋盘。

静如止水

主动 需场上有对手棋
  • 对方下一回合被跳过；你立即获得一次额外回合（该回合禁用技能）。

梅开二度

主动→准备 依赖：飞沙走石
  • 点击后进入准备阶段，给予对手 3 秒“擒拿”窗口。
  • 若 3 秒内没人擒拿 ⇒ 结算：再飞一次对手上一手。

擒拿

反应 3 秒窗口 可多次（仅窗口内）
  • 仅在对方“梅开二度准备”的 3 秒内出现，点击可取消本次“梅开二度”。

调虎离山

反应 3 秒窗口 每人一次
  • 当“擒拿”成功后，进攻方获得 3 秒可点的“调虎离山”。
  • 效果：移除对手盘面上仍存在的最近两子（若不足两子，则移除实际存在的那几颗）。

力拔山兮

主动 绝杀流程
  • 点击后给对手3 秒选择窗口（显示“捡起棋盘 / 手刀”，仅显示未用过者）。
  • 若 3 秒内无人选择 ⇒ 立刻获胜。

东山再起（按钮名：捡起棋盘）

反制 3 秒内点 10 秒口令 一次性
  • 点击后弹出输入框，10 秒内输入“东山再起”并发送 ⇒ 完全复原到力拔前的棋面与回合。
  • 台词：B“我东山再起” → A“什么，你竟然创造新词，那可是最高的奥！义！” → B“我只是参加过九年义务教！育！”。

手刀

反制 3 秒内点 10 秒口令 一次性
  • 点击后弹出输入框，10 秒内输入 see you again（大小写不敏感）并发送 ⇒ 先复原棋面，再对进攻方施加“静如止水”。
  • 台词：B“see you again～”，A“啊——啊——APT、APT…”。

两极反转

反制 3 秒按钮 通常一次
  • 当防守方已用完“东山再起”和“手刀”，且进攻方再次使用力拔山兮时，防守方技能区出现“两极反转”按钮（3 秒内可点）。
  • 效果：棋盘上双方棋子阵营互换（黑↔白），并封印进攻方的“力拔山兮”。随后立刻重算胜负。
  • 台词：B“揭开你的黑历史，改变你的战斗力！”；A“我竟然还是赢不了你…教练，让您蒙羞了！”。

AI 行为（普通难度默认）

  • 主动技能：60% 使用（择收益最高），40% 直接落子。
  • 擒拿：70%；调虎离山：75%。
  • 力拔反制：东山/手刀 90%（优先手刀），两极反转 80%。
  • “必胜/必防（盘面）”有温和兜底（少量失误率，保留人味儿）。
